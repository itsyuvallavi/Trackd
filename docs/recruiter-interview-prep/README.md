# Recruiter interview prep

These notes are for a **first conversation with a recruiter** (relationship and experience), not a deep technical interview.

**How to use this folder**

1. Read `01-recruiter-conversation-guide.md` once — it sets tone and structure.
2. Skim `03-tech-stack-explained.md` and `04-apis-and-backend-basics.md` so terms feel familiar.
3. Rehearse out loud from `02-your-experience-scripts.md` — that is what you actually say.

If a term confuses you during prep, check `05-terms-and-jargon-decoded.md`.

If your process is with **Atlana**, read `07-atlana-company-brief.md` — it explains **nearshore / TaaS** and what to clarify on the call.
